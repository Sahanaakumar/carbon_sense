# 🌍 CarbonSense

AI-powered Carbon Emission Forecasting System that predicts future CO₂ levels and helps in environmental decision-making.

---

## 📌 Problem Statement

Climate change is a major global issue. Governments and organizations lack simple tools to predict future carbon emissions and take preventive action.

---

## 💡 Solution

CarbonSense uses Machine Learning to:
- Analyze historical CO₂ emission data
- Predict future emissions (2026–2030)
- Provide insights through APIs for dashboards

---

## 🚀 Features

✅ Carbon emission forecasting  
✅ Future prediction (2026–2030)  
✅ REST API using FastAPI  
✅ Ready for dashboard integration  
✅ Scalable architecture  

---

## 🧠 Tech Stack

- Python 🐍  
- FastAPI ⚡  
- Prophet (Time Series Forecasting) 📊  
- Pandas & NumPy  

---

## 📊 Dataset

- Time-series CO₂ dataset (1950–2026)
- Cleaned and formatted for forecasting
